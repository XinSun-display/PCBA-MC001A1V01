### VS Code + ESP-IDF

1. **Install ESP-IDF Extension**
   - Search and install "ESP-IDF" extension in VS Code Extensions
   - Prerequisites: https://docs.espressif.com/projects/idf-im-ui/en/latest/prerequisites.html
   - ESP-IDF Extension for VS Code: https://github.com/espressif/vscode-esp-idf-extension/blob/master/README.md

2. **Configure ESP-IDF Environment**
   - Press `Ctrl+Shift+P` to open command palette
   - Type "ESP-IDF: Open ESP-IDF Installation Manager"
   - Select New Installation
   - Select Custom Installation
   - Target device: All (must include esp32s3)
   - Select ESP-IDF version: **5.5.3**
   - Set ESP-IDF installation path
   - ESP-IDF: https://docs.espressif.com/projects/vscode-esp-idf-extension/en/latest/installation.html

3. **Open Project**
   - Use "File > Open Folder" to open this project folder
   - VS Code will automatically recognize it as an ESP-IDF project

### Version Information

- **ESP-IDF**: 5.5.3
- **LVGL**: 9.1.0

### UI Modification Guide

If you want to modify the user interface, simply edit the `ui_init()` function in the `components/display/display.c` file.

#### Modification Steps:

1. Open the `components/display/display.c` file
2. Find the following code in the `display_init()` function:
   ```c
   // ui_demo();
   ui_init();
   ```
3. You can choose:
   - **Use default demo UI**: Uncomment `ui_demo()` and comment out `ui_init()`
     ```c
     ui_demo();
     // ui_init();
     ```
   - **Use custom UI**: Keep `ui_init()` and define your interface in `main/ui/ui.c`
   - **Create new UI function**: Reference the `ui_demo()` implementation to create your own UI function

#### UI Function Description:

- `ui_demo()`: Provides a simple demo interface with labels and buttons
- `ui_init()`: Loads the complete UI interface generated by SquareLine Studio

#### Notes:

- When modifying UI, ensure it's done between `lvgl_port_lock(0)` and `lvgl_port_unlock()`
- For more complex UI designs, it's recommended to use SquareLine Studio tool for interface design

### Use SquareLine Studio 

SquareLine Studio is a professional UI design tool that allows you to design user interfaces through drag-and-drop functionality.

#### Installing SquareLine Studio

1. Visit the [SquareLine Studio official website](https://squareline.io/) to download and install
2. Register an account and launch the software

#### Creating a New Project

1. **Open SquareLine Studio**
2. **Create New Project**:
   - Select "Create New Project"
   - Choose LVGL version 9.1
   - Select "Espressif" as the target platform
   - Choose "ESP-BOX"
   - Set project name and path
3. **Configure Display Settings**:
   - Resolution: 800 x 480 (according to your display specifications)
   - Color depth: 16 bit
   - LVGL version: 9.1.0

#### Designing UI Interface

1. **Adding Components**:
   - Drag components from the left toolbar to the canvas
2. **Setting Properties**:
   - Adjust component position, size, color, etc. in the right properties panel
3. **Adding Events**:
   - Select a component and add event handlers in the Events tab

#### Exporting Code

1. **Export Settings**:
   - Click "Export" → "Export UI Files"
   - Select export path as the project's `main/ui/` folder
2. **Export Files**:
   - `ui.c` and `ui.h`: Main UI code
   - Other related screen and component files

#### Integrating into Project

1. **Copy Files**:
   - Copy the exported `ui.c` and `ui.h` files to the `main/ui/` folder
2. **Call UI**:
   - Call the `ui_init()` function in display_init() in `components/display/display.c`
